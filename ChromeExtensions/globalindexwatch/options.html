Global Index Watch - Chrome Extension - Help

Thank you for trying out my Chrome Extension. Global Index Watch is an extension that is created on top of the iGoogle gadget Global Index Watch which is an extremely simple module that tracks the indices of most of the markets in North America, Latin America, Europe, Asia and Australia.


You can customize the extension by choosing the options on this page.
